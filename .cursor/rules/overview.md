# Project Overview

This is a React 19 + TypeScript onboarding wizard application using Vite, Zustand, React Query, and React Router.

## Tech Stack

- **React**: 19.0.0
- **TypeScript**: 5.6.3
- **State Management**: Zustand 5.0.9 with persist middleware
- **Data Fetching**: React Query (@tanstack/react-query) 5.59.0
- **Routing**: React Router DOM 7.1.1
- **Validation**: Zod 3.23.8
- **Styling**: Tailwind CSS 3.4.14
- **Testing**: Vitest 4.0.15 with Testing Library
- **Build Tool**: Vite 5.4.8
